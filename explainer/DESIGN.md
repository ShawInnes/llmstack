## Style Prompt

Data Drift — inspired by Refik Anadol. Deep black canvas with electric purple and cyan accents. Futuristic, immersive, weightless. Text floats in space. Radial glows pulse like data breathing. Infrastructure visualized as living topology, not static boxes. Fluid transitions blur between scenes like shifting focus through a neural network.

## Colors

| Role       | Hex       | Usage                              |
| ---------- | --------- | ---------------------------------- |
| Background | `#0a0a0a` | All scene backgrounds              |
| Foreground | `#e4e4f0` | Primary text, purple-tinted white  |
| Accent 1   | `#7c3aed` | Purple — headlines, glows, borders |
| Accent 2   | `#06b6d4` | Cyan — data, stats, highlights     |
| Dim        | `#64648a` | Labels, secondary text             |

## Typography

- **Display:** Space Grotesk 700-900 — geometric, techy, confident
- **Mono:** JetBrains Mono 400 — code, ports, commands, data labels

## Motion

- Eases: `sine.inOut`, `power2.out`, `power3.out` — smooth, continuous, organic
- Entrances: 0.4-0.7s, varied directions (y, x, scale, opacity)
- Ambient: radial glows breathing via slow scale oscillation
- Transitions: blur crossfade (primary), color dip to black (accent)

## What NOT to Do

- No sharp snaps or bounce eases — everything fluid
- No gradient text (`background-clip: text`)
- No cyan-on-dark cliche — use cyan sparingly for data only
- No uniform card grids — vary element sizes and spacing
- No static backgrounds — every scene has ambient motion
